HTML Cheatsheet

a - Hyperlink

A hyperlink consists of 2 basic elements, a string of text and another resource (a document or webpage for instance). It allows users to click it and access the linked resource.

Example

    
        <a href="https://developer.mozilla.org/"> Mozilla Developer Network </a>
Mozilla Developer Network
