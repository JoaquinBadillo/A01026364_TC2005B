HTML Cheatsheet

a - Hyperlink

A hyperlink consists of 2 basic elements, a string of text and another resource (a document or webpage for instance). It allows users to click it and access the linked resource.

Example

Mozilla Developer Network 

  <!-- HTML -->
  <a href="https://developer.mozilla.org/">  
      Mozilla Developer Network
  </a>

p - Paragraph

Paragraphs are elements that contain text and are separated by whitespaces when used in succession.

Example

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Repellendus sequi voluptatibus magnam hic odio nisi rem consectetur libero nobis voluptate quibusdam ipsum repudiandae reiciendis harum, nam ducimus temporibus dignissimos facilis! 


  <!-- HTML -->
  <p>
    Lorem ipsum dolor sit amet consectetur, adipisicing elit. Repellendus sequi  
    voluptatibus magnam hic odio nisi rem consectetur libero nobis voluptate  
    quibusdam ipsum repudiandae reiciendis harum, nam ducimus temporibus dignissimos  
    facilis!
  </p>

h1, h2, ... , h6 - Headings

Example

This is a level 4 heading


  <!-- HTML -->
  <h4>
    This is a level 4 heading  
  </h4>

div - Division

Creates a "division", which can be understood as section or container for styling.

Example

Help me! I'm inside a div. 

  <!-- HTML -->
  <div>
      Help me! I'm inside a div.  
  </div>

span - Span

Spans are inline elements that allow you to group and style content.

Example

Hello, I will use a span to add style.


  <!-- HTML -->
  <p>
    Hello, I will use a <span style="color: #E28964">span</span> to add style.  
  </p>

strong - Strong

The visual effect of the strong tag is to make the text bold. However, it is used semantically to indicate that the bolded text has strong importance.

Example

Important: Tags can have semantic meaning. 

  <!-- HTML -->
  <strong>Important</strong>: Tags can have semantic meaning.

i - Italicize

The i tag will italicize text. However, it is used semantically to convey that text should be read with a different tone. When trying to emphasize with italics the em tag should be used.

Example

Its Leviosa, not Leviosaaa 

  <!-- HTML -->
  Its <i>Leviosa</i>, not <i>Leviosaaa</i>

u - Underline

This tag will underline text. It is used semantically for annotations like annotate spelling errors.

Example

I lov syntax highlighters! 

  <!-- HTML -->
  I <u style="text-decoration:#f00 wavy underline">lov</u> syntax highlighters!

b - Bold

This tag, as mentioned previously, makes the text bold; but without additional importance. That is it makes the text stand out but only stylistically (not semanitcally).

Example

Poor b tag, it was left behind by the semantic tags. 

  <!-- HTML -->
  Poor b tag, it was left behind by the <b>semantic</b> tags.

strike - Strikethrough

Strike is deprecated in HTML5 and it was used to cross out (strikethrough) text. It has been replaced by the semantic del and s tags.

Example

Research about the strike tag 

  <!-- HTML -->
  <strike>Research about the strike tag</strike>

small - Small

Makes text smaller and is mainly used to add side-comments or small print.

Example

Some license information 

  <!-- HTML -->
    <small>Some license information</small>

em - Emphasize

Used semantically to emphasize text with italics.

Example

I need more time! 

  <!-- HTML -->
  I <em>need</em> more time!

header - Header

Used to add introductory content like titles and authors for articles.

Example

Real fake doors

Come get fake doors 

  <!-- HTML -->
  <header>
    <h4>Real fake doors</h4>  

    Come get fake doors
  </header>

section - Section

Sections are divisions (divs) with additional semantic meaning. A section is considered to be standalone and are usually expected to have headings.

Example

Section Example

This is a very important section! 

  <!-- HTML -->
  <section>
    <h4>Section Example</h4>
    
    This is a very important section!  
  </section>

article - Article

Articles are used for blog entries, news articles, product cards or in general self-contained content.

Example

Chat GPT is scary

Chat GPT, Open AI's chatbot is likely going to affect many jobs


  <!-- HTML -->
  <article>
    <h4>Chat GPT is scary</h4>
    
    <p>Chat GPT, Open AI's chatbot is likely going to affect many jobs</p>  
  </article>
